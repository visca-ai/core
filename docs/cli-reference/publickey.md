# publickey

Output your visca public key used for Git operations

Aliases:

- pubkey

## Usage

```console
visca publickey [flags]
```

## Options

### --reset

|      |                   |
| ---- | ----------------- |
| Type | <code>bool</code> |

Regenerate your public key. This will require updating the key on any services it's registered with.

### -y, --yes

|      |                   |
| ---- | ----------------- |
| Type | <code>bool</code> |

Bypass prompts.
