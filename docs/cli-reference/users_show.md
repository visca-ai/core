# users show

Show a single user. Use 'me' to indicate the currently authenticated user.

## Usage

```console
visca users show [flags] <username|user_id|'me'>
```

## Description

```console
 $ visca users show me
```

## Options

### -o, --output

|         |                          |
| ------- | ------------------------ |
| Type    | <code>table\|json</code> |
| Default | <code>table</code>       |

Output format.
